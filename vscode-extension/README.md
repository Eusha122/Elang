## üöÄ Getting Started

### Prerequisites

This extension requires the **Elang Runtime** to be installed on your system.
1. Download the installer from the [Official Repository](https://github.com/Eusha122/Elang).
2. Ensure the `elang` command is available in your system `PATH`.

### Installation

- **From the Installer**: The [Elang Professional Installer](https://github.com/Eusha122/Elang/tree/main/installer) will automatically install this extension if VS Code is detected.
- **Manual Side-load**: Install via the [VSIX package](https://github.com/Eusha122/Elang/tree/main/vscode-extension) using the VS Code "Install from VSIX..." command.

## üõ† Usage

1. **Open** any file with the `.elang` extension.
2. **Execute** your code via:
   - The **‚ñ∂ Play** button in the top-right editor bar.
   - Right-click anywhere in the editor ‚Üí **Run Elang File**.
   - Pressing **`Ctrl+Shift+R`**.
